Preliminary Tests on Giant Components in Random Graphs

The python code for generating the above video ishere.

This page is a preliminary computational test of some results discussed in the paper on sparse random graphs around the critical regime. I am not trying to re-prove the paper. I am trying to see, in a concrete and sequential way, what the graph is doing near m \approx n/2, and how far a direct simulation can go before the asymptotic theory begins to ask for more than raw numerics.

Paper: Janson, Knuth, Łuczak, and Pittel, The Birth of the Giant Component
The particular probabilities I am looking at here are the ones around the sparse critical regime, including the event that all components are tree / unicyclic / bicyclic, and the event that there is at most one “other” component.
Scope. This is only a preliminary test on the results of the paper. One script shows a single evolving sample path. The other script runs many samples and compares empirical frequencies against the theoretical reference values. That is useful, but it is still not the full asymptotic argument.

What I am trying to do

  • Start with an empty graph on n labelled vertices.
  • Add edges one at a time, so the evolution is visible rather than hidden in occasional snapshots.
  • Track whether the connected components are trees, unicyclic, bicyclic, or “other”.
  • Look especially at the regime m = n/2, since that is where the paper states sharp limiting behavior.
  • Use one animation for intuition and one large-run simulation for empirical comparison.

Why I separated the page into two scripts

I do not want one bloated script that tries to do everything at once. One task is visual and sequential. The other is statistical and repetitive. Those are different jobs.

  • Animation script: useful for seeing one path evolve edge by edge.
  • Simulation script: useful for seeing whether the abstract’s probabilities show up empirically when the number of runs is large.

1. Animation of one evolving graph

This script follows a single run. It places the vertices in a circular layout, numbers them in small fonts, adds one edge at a time, and shows the component counts at the bottom. Toward the end, it also reminds the viewer that one animation is only one sample path, not an estimate of the limiting probabilities.

Suggested filename in the same GitHub Pages folder: critical_graph_evolution.mp4 or critical_graph_evolution.gif.
giant_component_animation.py
Exact animation script for one run
#!/usr/bin/env python3

import random
from dataclasses import dataclass
from typing import Optional

import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
import networkx as nx
import pandas as pd


# ----------------------------
# Component classification
# ----------------------------

def classify_component(subgraph: nx.Graph) -> str:
    v = subgraph.number_of_nodes()
    e = subgraph.number_of_edges()

    if e == v - 1:
        return "tree"
    elif e == v:
        return "unicyclic"
    elif e == v + 1:
        return "bicyclic"
    else:
        return "other"


def component_type_counts(G: nx.Graph) -> dict:
    comps = [G.subgraph(c).copy() for c in nx.connected_components(G)]

    counts = {
        "tree": 0,
        "unicyclic": 0,
        "bicyclic": 0,
        "other": 0,
    }

    for comp in comps:
        counts[classify_component(comp)] += 1

    return counts


def graph_stats(G: nx.Graph) -> dict:
    comps = [G.subgraph(c).copy() for c in nx.connected_components(G)]

    largest_component_size = max((c.number_of_nodes() for c in comps), default=0)
    isolated_vertices = sum(1 for _, d in G.degree() if d == 0)

    type_counts = component_type_counts(G)

    all_in_t_u_b = (type_counts["other"] == 0)
    at_most_one_other = (type_counts["other"] <= 1)

    return {
        "nodes": G.number_of_nodes(),
        "edges": G.number_of_edges(),
        "components": len(comps),
        "largest_component": largest_component_size,
        "isolated_vertices": isolated_vertices,
        "trees": type_counts["tree"],
        "unicyclic": type_counts["unicyclic"],
        "bicyclic": type_counts["bicyclic"],
        "other": type_counts["other"],
        "all_tree_unicyclic_bicyclic": int(all_in_t_u_b),
        "at_most_one_other_component": int(at_most_one_other),
    }


# ----------------------------
# Random graph process
# ----------------------------

def build_edge_order(n: int, rng: random.Random):
    all_edges = [(i, j) for i in range(n) for j in range(i + 1, n)]
    rng.shuffle(all_edges)
    return all_edges


def run_one_process(
    n: int,
    max_edges: int,
    seed: Optional[int] = None,
    keep_edge_sequence: bool = True,
) -> pd.DataFrame:
    rng = random.Random(seed)

    G = nx.Graph()
    G.add_nodes_from(range(n))

    edge_order = build_edge_order(n, rng)[:max_edges]

    rows = []

    rows.append({
        "step": 0,
        "added_u": None,
        "added_v": None,
        **graph_stats(G),
    })

    for step, (u, v) in enumerate(edge_order, start=1):
        G.add_edge(u, v)
        rows.append({
            "step": step,
            "added_u": u if keep_edge_sequence else None,
            "added_v": v if keep_edge_sequence else None,
            **graph_stats(G),
        })

    return pd.DataFrame(rows)


# ----------------------------
# Animation
# ----------------------------

def animate_one_process(
    n: int = 50,
    max_edges: Optional[int] = None,
    seed: Optional[int] = 20260330,
    interval_ms: int = 1000,
    outfile: Optional[str] = "critical_graph_evolution.gif",
    show_labels: bool = True,
):
    """
    Animate a single random graph process with:
    - circular node layout
    - small numbered labels
    - bottom stats panel
    - theorem-reference panel near the end
    """
    import math

    if max_edges is None:
        max_edges = n // 2

    history = run_one_process(
        n=n,
        max_edges=max_edges,
        seed=seed,
        keep_edge_sequence=True,
    )

    edge_sequence = []
    for _, row in history.iloc[1:].iterrows():
        edge_sequence.append((int(row["added_u"]), int(row["added_v"])))

    # Circular layout
    G0 = nx.Graph()
    G0.add_nodes_from(range(n))
    pos = nx.circular_layout(G0)

    fig = plt.figure(figsize=(13, 8.5))
    gs = fig.add_gridspec(2, 1, height_ratios=[4.8, 2.2])

    ax_graph = fig.add_subplot(gs[0])
    ax_info = fig.add_subplot(gs[1])

    theory_tub = math.sqrt(2 / 3) * math.cosh(math.sqrt(5 / 18))   # about 0.9325
    theory_one_other = 0.9957

    def verdict_text(row, near_end: bool) -> str:
        tub_now = int(row["all_tree_unicyclic_bicyclic"])
        one_other_now = int(row["at_most_one_other_component"])

        verdict1 = (
            "YES" if tub_now == 1 else "NO"
        )
        verdict2 = (
            "YES" if one_other_now == 1 else "NO"
        )

        lines = []
        lines.append("Interpretation relative to the abstract:")
        lines.append(
            f"- Entirely tree/unicyclic/bicyclic components right now? {verdict1}"
        )
        lines.append(
            f"- Trees/unicyclic + at most one other component right now? {verdict2}"
        )

        if near_end:
            lines.append("")
            lines.append("Limiting reference values from the theorem:")
            lines.append(
                f"- P(all components are tree/unicyclic/bicyclic) ≈ {theory_tub:.4f}"
            )
            lines.append(
                f"- P(trees/unicyclic + at most one other component) ≈ {theory_one_other:.4f}"
            )
            lines.append("")
            lines.append(
                "This animation is one sample path only, not an empirical estimate of those probabilities."
            )

        return "\n".join(lines)

    def format_info(row, frame: int, total_frames: int) -> str:
        added_u = row["added_u"]
        added_v = row["added_v"]

        if pd.isna(added_u) or pd.isna(added_v):
            edge_text = "Added edge: none"
        else:
            edge_text = f"Added edge: ({int(added_u)}, {int(added_v)})"

        line1 = (
            f"Step: {int(row['step'])}/{total_frames - 1}    "
            f"Edges: {int(row['edges'])}    "
            f"Components: {int(row['components'])}    "
            f"Largest component: {int(row['largest_component'])}"
        )

        line2 = (
            f"Isolated vertices: {int(row['isolated_vertices'])}    "
            f"Trees: {int(row['trees'])}    "
            f"Unicyclic: {int(row['unicyclic'])}    "
            f"Bicyclic: {int(row['bicyclic'])}    "
            f"Other: {int(row['other'])}"
        )

        line3 = (
            f"All tree/unicyclic/bicyclic: {int(row['all_tree_unicyclic_bicyclic'])}    "
            f"At most one other component: {int(row['at_most_one_other_component'])}"
        )

        near_end = frame >= total_frames - 5
        line4 = verdict_text(row, near_end=near_end)

        return f"{edge_text}\n{line1}\n{line2}\n{line3}\n\n{line4}"

    def update(frame: int):
        ax_graph.clear()
        ax_info.clear()

        G = nx.Graph()
        G.add_nodes_from(range(n))
        G.add_edges_from(edge_sequence[:frame])

        # Smaller dots; circular layout; numbered nodes
        node_size = 70 if n <= 80 else 35
        font_size = 6 if n <= 80 else 4

        nx.draw_networkx_nodes(
            G,
            pos,
            ax=ax_graph,
            node_size=node_size,
        )
        nx.draw_networkx_edges(
            G,
            pos,
            ax=ax_graph,
            width=0.9,
        )

        if show_labels:
            labels = {i: str(i) for i in range(n)}
            nx.draw_networkx_labels(
                G,
                pos,
                labels=labels,
                ax=ax_graph,
                font_size=font_size,
                verticalalignment="center",
                horizontalalignment="center",
            )

        ax_graph.set_title(f"Random graph evolution on n = {n}", fontsize=14)
        ax_graph.set_axis_off()

        row = history.iloc[frame]
        info_text = format_info(row, frame=frame, total_frames=len(history))

        ax_info.set_axis_off()
        ax_info.text(
            0.01,
            0.98,
            info_text,
            ha="left",
            va="top",
            fontsize=10,
            family="monospace",
        )

    anim = FuncAnimation(
        fig,
        update,
        frames=len(history),
        interval=interval_ms,
        repeat=False
    )

    if outfile:
        fps = max(1, round(1000 / interval_ms))

        if outfile.lower().endswith(".gif"):
            anim.save(outfile, writer="pillow", fps=fps)
        elif outfile.lower().endswith(".mp4"):
            try:
                anim.save(outfile, writer="ffmpeg", fps=fps)
            except Exception:
                fallback = outfile[:-4] + ".gif"
                print("ffmpeg unavailable; saving GIF instead.")
                anim.save(fallback, writer="pillow", fps=fps)
                print(f"Saved animation to: {fallback}")
            else:
                print(f"Saved animation to: {outfile}")
        else:
            raise ValueError("outfile must end with .gif or .mp4")

    plt.tight_layout()
    plt.show()


if __name__ == "__main__":
    animate_one_process(
        n=50,                       # try 30, 50, or 80 first
        max_edges=25,               # or n//2
        seed=20260330,
        interval_ms=1000,           # 1 second pause
        outfile="critical_graph_evolution.mp4",
        show_labels=False
    )

2. Large-run simulation at m = n/2

This script is the empirical side. It repeats the process many times, records the component structure, and then reports the observed frequency of the two events that I care about:

  • all components are tree / unicyclic / bicyclic,
  • trees / unicyclic plus at most one other component.

It also saves the outputs as CSV and Excel files, which is useful if I later want to plot finite-size behavior for n = 200, 500, 1000, ... rather than stare only at terminal output.

import math
import os
import random
from dataclasses import dataclass
from typing import Optional

import networkx as nx
import pandas as pd


# ----------------------------
# Component classification
# ----------------------------

def classify_component(subgraph: nx.Graph) -> str:
    v = subgraph.number_of_nodes()
    e = subgraph.number_of_edges()

    if e == v - 1:
        return "tree"
    elif e == v:
        return "unicyclic"
    elif e == v + 1:
        return "bicyclic"
    else:
        return "other"


def component_type_counts(G: nx.Graph) -> dict:
    comps = [G.subgraph(c).copy() for c in nx.connected_components(G)]

    counts = {
        "tree": 0,
        "unicyclic": 0,
        "bicyclic": 0,
        "other": 0,
    }

    for comp in comps:
        counts[classify_component(comp)] += 1

    return counts


def graph_stats(G: nx.Graph) -> dict:
    comps = [G.subgraph(c).copy() for c in nx.connected_components(G)]

    largest_component_size = max((c.number_of_nodes() for c in comps), default=0)
    isolated_vertices = sum(1 for _, d in G.degree() if d == 0)

    type_counts = component_type_counts(G)

    all_in_t_u_b = (type_counts["other"] == 0)
    at_most_one_other = (type_counts["other"] <= 1)

    return {
        "nodes": G.number_of_nodes(),
        "edges": G.number_of_edges(),
        "components": len(comps),
        "largest_component": largest_component_size,
        "isolated_vertices": isolated_vertices,
        "trees": type_counts["tree"],
        "unicyclic": type_counts["unicyclic"],
        "bicyclic": type_counts["bicyclic"],
        "other": type_counts["other"],
        "all_tree_unicyclic_bicyclic": int(all_in_t_u_b),
        "at_most_one_other_component": int(at_most_one_other),
    }


# ----------------------------
# Random graph process
# ----------------------------

def build_edge_order(n: int, rng: random.Random):
    all_edges = [(i, j) for i in range(n) for j in range(i + 1, n)]
    rng.shuffle(all_edges)
    return all_edges


def run_one_process(
    n: int,
    max_edges: int,
    seed: Optional[int] = None,
    keep_edge_sequence: bool = True,
) -> pd.DataFrame:
    rng = random.Random(seed)

    G = nx.Graph()
    G.add_nodes_from(range(n))

    edge_order = build_edge_order(n, rng)[:max_edges]

    rows = []

    # Step 0
    rows.append({
        "step": 0,
        "added_u": None,
        "added_v": None,
        **graph_stats(G),
    })

    for step, (u, v) in enumerate(edge_order, start=1):
        G.add_edge(u, v)
        rows.append({
            "step": step,
            "added_u": u if keep_edge_sequence else None,
            "added_v": v if keep_edge_sequence else None,
            **graph_stats(G),
        })

    return pd.DataFrame(rows)


@dataclass
class SimulationResult:
    raw_history: pd.DataFrame
    averages_by_step: pd.DataFrame
    report_step_rows: pd.DataFrame
    summary: pd.DataFrame


def run_many_processes(
    n: int,
    num_runs: int,
    report_edges: Optional[int] = None,
    record_full_history: bool = True,
    base_seed: Optional[int] = None,
    show_progress: bool = True,
) -> SimulationResult:
    if report_edges is None:
        report_edges = n // 2

    max_edges = report_edges if not record_full_history else report_edges

    master_rng = random.Random(base_seed)
    all_runs = []

    for run in range(1, num_runs + 1):
        run_seed = master_rng.randint(0, 10**12)

        df_run = run_one_process(
            n=n,
            max_edges=max_edges,
            seed=run_seed,
            keep_edge_sequence=True,
        )
        df_run.insert(0, "run", run)
        df_run.insert(1, "run_seed", run_seed)
        all_runs.append(df_run)

        if show_progress and (run == 1 or run % max(1, num_runs // 20) == 0 or run == num_runs):
            print(f"Completed run {run}/{num_runs}")

    raw_history = pd.concat(all_runs, ignore_index=True)

    # Step-by-step averages
    avg_cols = [
        "edges",
        "components",
        "largest_component",
        "isolated_vertices",
        "trees",
        "unicyclic",
        "bicyclic",
        "other",
        "all_tree_unicyclic_bicyclic",
        "at_most_one_other_component",
    ]

    averages_by_step = (
        raw_history.groupby("step", as_index=False)[avg_cols]
        .mean()
        .rename(columns={
            "edges": "avg_edges",
            "components": "avg_components",
            "largest_component": "avg_largest_component",
            "isolated_vertices": "avg_isolated_vertices",
            "trees": "avg_trees",
            "unicyclic": "avg_unicyclic",
            "bicyclic": "avg_bicyclic",
            "other": "avg_other",
            "all_tree_unicyclic_bicyclic": "prob_all_tree_unicyclic_bicyclic",
            "at_most_one_other_component": "prob_at_most_one_other_component",
        })
    )

    report_step_rows = raw_history[raw_history["step"] == report_edges].copy()

    p_hat_tub = report_step_rows["all_tree_unicyclic_bicyclic"].mean()
    p_hat_one_other = report_step_rows["at_most_one_other_component"].mean()

    theory_tub = math.sqrt(2 / 3) * math.cosh(math.sqrt(5 / 18))
    theory_one_other = 0.9957  # abstract's rounded value

    summary = pd.DataFrame([{
        "n": n,
        "num_runs": num_runs,
        "report_edges": report_edges,
        "empirical_prob_all_tree_unicyclic_bicyclic": p_hat_tub,
        "theory_prob_all_tree_unicyclic_bicyclic": theory_tub,
        "difference_all_tree_unicyclic_bicyclic": p_hat_tub - theory_tub,
        "empirical_prob_at_most_one_other_component": p_hat_one_other,
        "theory_prob_at_most_one_other_component": theory_one_other,
        "difference_at_most_one_other_component": p_hat_one_other - theory_one_other,
    }])

    return SimulationResult(
        raw_history=raw_history,
        averages_by_step=averages_by_step,
        report_step_rows=report_step_rows,
        summary=summary,
    )


def save_outputs(result: SimulationResult, output_dir: str = "critical_graph_outputs"):
    os.makedirs(output_dir, exist_ok=True)

    raw_csv = os.path.join(output_dir, "raw_history_all_runs.csv")
    avg_csv = os.path.join(output_dir, "averages_by_step.csv")
    report_csv = os.path.join(output_dir, "report_step_rows.csv")
    summary_csv = os.path.join(output_dir, "summary.csv")
    xlsx_path = os.path.join(output_dir, "critical_graph_outputs.xlsx")

    result.raw_history.to_csv(raw_csv, index=False)
    result.averages_by_step.to_csv(avg_csv, index=False)
    result.report_step_rows.to_csv(report_csv, index=False)
    result.summary.to_csv(summary_csv, index=False)

    with pd.ExcelWriter(xlsx_path, engine="openpyxl") as writer:
        result.raw_history.to_excel(writer, sheet_name="raw_history", index=False)
        result.averages_by_step.to_excel(writer, sheet_name="averages_by_step", index=False)
        result.report_step_rows.to_excel(writer, sheet_name="report_step_rows", index=False)
        result.summary.to_excel(writer, sheet_name="summary", index=False)

    print(f"Saved files to: {output_dir}")


if __name__ == "__main__":
    # Choose n large enough to start seeing asymptotics.
    # Try n = 200, 500, 1000, ... and compare.
    n = 200
    num_runs = 1000
    report_edges = n // 2
    base_seed = 20260330

    result = run_many_processes(
        n=n,
        num_runs=num_runs,
        report_edges=report_edges,
        record_full_history=True,
        base_seed=base_seed,
        show_progress=True,
    )

    print("\nSummary:")
    print(result.summary.to_string(index=False))

    print("\nFirst few rows at the reporting step:")
    print(result.report_step_rows.head(10).to_string(index=False))

    save_outputs(result, output_dir="critical_graph_outputs")

How I am reading the outputs

I am reading the page in two passes.

  • Pass 1: the animation tells me what one realization actually looks like edge by edge.
  • Pass 2: the repeated simulation tells me whether the paper’s reference values start showing up numerically when the number of runs is large.

That is the point of keeping both scripts here. One gives intuition. The other gives a rough empirical check. Neither one, by itself, is the theorem.
