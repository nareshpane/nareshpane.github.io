Step 5: Batch OCR for All Pages

Process the full document once the first test looks usable

Placeholder description: this section will later show how to batch-process many scanned pages into text files and keep the results organized cleanly.

Batch OCR loop
Run Tesseract across all page images
mkdir -p ocr_text
for img in pages/*.png; do
  base=$(basename "$img" .png)
  tesseract "$img" "ocr_text/$base" -l script/Devanagari
done
